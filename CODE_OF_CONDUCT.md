# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:
- Being respectful and constructive
- Giving and receiving feedback gracefully
- Focusing on what is best for the project and users

Examples of unacceptable behavior:
- Harassment, hate speech, or personal attacks
- Trolling, insulting, or derogatory comments
- Publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards.
They may remove, edit, or reject comments, commits, issues, or other contributions that violate this Code of Conduct.

## Reporting

Report incidents via GitHub issues marked `confidential` request or by contacting the maintainer directly.

This Code of Conduct is adapted from the Contributor Covenant, version 2.1:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/
